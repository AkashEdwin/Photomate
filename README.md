# Photomate
[![Gem Version]
[![Gem Downloads]

An image sharing website.

## Requirements

Ruby 2.4.4

Rails 5.1.4

## How to use

open cmd
>git clon https://github.com/AkashEdwin/Photomate.git


cd photomate

bundle install    --  to install the gems

rails s    -- to start the server

The server should be running in localhost:3000 in browser.

Enjoy!
